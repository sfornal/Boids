Boids

My attempt at recreating Boids using Canvas and JavaScript. This is purely for my own learning.

Download this project as a .zip file Download this project as a tar.gz file

View the completed project at http://boids.mrfornal.net.

The purpose of this project is two-fold:

  1. Teach myself JavaScript, using an algorithm I already (mostly) know.
  2. Have the results be interesting, and not just a crapy "Hello, World"-ish demo.

Hopefully I will learn quite a bit along the way. Thanks for looking!
